If the panel says “refused to connect” The iframe loads the app from port 5173 (Vite). That process must be running: npm run dev in this repo. Then open http://127.0.0.1:5173/widget-standalone-test.html (same server as the iframe — simplest). If you use Live Server on :5500 instead, keep Vite running in another terminal and ensure base-url below matches the URL that loads the app in a new tab.
